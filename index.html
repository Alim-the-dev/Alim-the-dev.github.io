Random Color Generator Project – HTML, CSS, JavaScript

This project features a dynamic tool that generates up to 30 unique random colors using JavaScript. Each time the page is refreshed, a new set of colors and their hexadecimal codes are displayed. The layout is fully responsive — color blocks are arranged in four columns on larger screens, scaling down to three, two, and then a single column on smaller devices to ensure optimal usability across all screen sizes.

Here are some screenshots showcasing the project in action:

Technologies Used
  • HTML5 – Provided the foundational structure of the webpage.
  • CSS3 – Styled the layout and elements with a clean, modern look.
  • JavaScript – Generated and displayed the random color codes dynamically.
Role and Contributions
  • Front-End Development: Designed and built the HTML structure and used CSS to ensure a clean and visually pleasing layout.
  • Responsive Layout: Implemented responsive design using CSS media queries for optimal display on desktops, tablets, and mobile devices.
  • JavaScript Logic: Developed functions to generate random hex codes and apply them to each container dynamically.
  • User Experience: Ensured that each color block displays its corresponding code for easy reference and usability.

Try out the Random Color Generator live or dive into the GitHub repository to explore the codebase in detail.
